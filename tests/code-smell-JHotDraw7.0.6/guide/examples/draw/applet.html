This demo applet shows a diagram, which I made with the Draw Demo Application.

You can move figures on the drawing area using the mouse. When you select a figure, handles appear, which allow to manipulate the figure. Scribbles and Polygon figures provide multiple levels of handles. Double click on a figure, to ascend to a higher level of handles. To open a context menu on the drawing area or on a figure, press the right mouse button (on a single-button mouse, keep the mouse button pressed for one second).

The first toolbar at the bottom of the applet contains the creation tools. Select a creation tool, and then press the mouse button (and drag) on the drawing area to create a figure. The cogwheel on the right opens a popup menu.

The second toolbar at the bottom of the applet contains attribute tools. Select a figure in the drawing area, and then use an attribute tool, to change the appearance of a figure.
